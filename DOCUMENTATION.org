* Nowo Language

Nowo is a small, functional programming language with dynamic typing,
first-class functions, closures, and array support.

** Running

#+begin_src sh
go run . example.nowo
go test ./...
#+end_src

** Syntax

*** Literals

| Example     | Type   |
|-------------+--------|
| =42=        | Number |
| =3.14=      | Number |
| =nil=       | Nil    |
| =.[1, 2, 3]= | Array  |
| =.[]=       | Empty array |

*** Identifiers and declarations

#+begin_src nowo
x := 42           -- declare and bind
x = 10            -- reassign
#+end_src

*** Arithmetic

=+= =-= =*= =/= work as expected. =++= concatenates arrays.

#+begin_src nowo
1 + 2 * 3         -- => 7 (standard precedence)
{1 + 2} * 3       -- => 9 (grouping with braces)
#+end_src

*** Comparison

==== ==!= =<= =>= =<== =>== all return 1 (true) or 0 (false).

*** Functions

#+begin_src nowo
inc := (x) x + 1
add := (a, b) a + b
apply := (f, x) f(x)

add(1, 2)          -- => 3
inc(41)            -- => 42
#+end_src

Functions with default parameters:

#+begin_src nowo
f := (x, y = 10, z = y + 1) x + y + z
f(1)               -- => 1 + 10 + 11 = 22
f(1, 2)            -- => 1 + 2 + 3 = 6
f(1, 2, 3)         -- => 1 + 2 + 3 = 6
#+end_src

*** Conditionals (ternary)

#+begin_src nowo
x == 1 ? "yes" ! "no"
x > 0 ? x ! -x
#+end_src

*** Blocks and sequencing

Braces group expressions. =;= sequences them (last value is returned).

#+begin_src nowo
{ 1; 2; 3 }        -- => 3
#+end_src

*** Pipe operator

=expr : fn(args)= becomes =fn(expr, args)=.

#+begin_src nowo
items :map((x) x + 1) :filter(iseven) :reduce(add, 0)
#+end_src

*** Arrays

#+begin_src nowo
arr := .[10, 20, 30]
arr[1]             -- => 20 (zero-based)
arr[-1]            -- => 30 (negative wraps)
#arr               -- => 3 (length)
a := .[1, 2]
b := .[3, 4]
a ++ b             -- => .[1, 2, 3, 4]
#+end_src

=arr[i] = val= assigns to an element (zero-based, supports negative indices):

#+begin_src nowo
arr := .[10, 20, 30]
arr[1] = 99          -- => .[10, 99, 30]
arr[-1] = 0          -- => .[10, 99, 0]
#+end_src

** Pattern matching

The =|= operator destructures a value against one or more patterns. Each arm consists
of a pattern and a body separated by =->=. The first matching arm's body is evaluated
and returned. If no arm matches, the expression evaluates to =nil=.

#+begin_src nowo
42 | 42 -> "matched", _ -> "no"
-- => "matched"
#+end_src

Arms are separated by commas:

#+begin_src nowo
0 | 1 -> "one", 0 -> "zero"
-- => "zero"
#+end_src

*** Literal patterns

Match a specific number:

#+begin_src nowo
n := 3
n | 1 -> "small", 2 -> "medium", 3 -> "large"
-- => "large"
#+end_src

*** Bind patterns

Capture the value into a variable (always matches):

#+begin_src nowo
42 | x -> x + 1
-- => 43
#+end_src

*** Array patterns

Destructure arrays by position. The number of elements must match exactly:

#+begin_src nowo
.[1, 2, 3] | .[a, b, c] -> a + b + c
-- => 6
#+end_src

Use =*name= to capture the remaining elements (array length must be at least
the number of fixed elements):

#+begin_src nowo
.[1, 2, 3] | .[x, *rest] -> x + #rest
-- => 3

.[1] | .[x, *rest] -> x + #rest
-- => 1
#+end_src

Empty array pattern:

#+begin_src nowo
.[] | .[] -> "empty", _ -> "not empty"
-- => "empty"
#+end_src

*** Nested patterns

Array patterns can contain other patterns, including nested array patterns:

#+begin_src nowo
.[1, .[2, 3]] | .[a, .[b, c]] -> a + b + c
-- => 6
#+end_src

** Built-ins

- =nil= -- The nil value (can be shadowed).
- =print(...)= -- Prints arguments to stdout, returns nil.

** Truthiness

=nil= and =0= are falsy. Everything else is truthy.

** Comments

=--= starts a line comment.

#+begin_src nowo
-- this is a comment
x := 1 -- inline comment
#+end_src

** Emacs Mode

File =nowo.el= provides =nowo-mode= with syntax highlighting, indentation,
comment support, and imenu. Auto-loads for =.nowo= files.
